# SIMPLEX CNC Cost Calculator

A comprehensive React application for calculating the monthly and annual operating costs of CNC machines. This tool helps business owners estimate expenses related to electricity, maintenance, and consumables.

## Features

- **Multi-language Support**: Switch between English and Arabic.
- **Multi-currency**: Support for USD, SAR, and EGP.
- **Visual Breakdown**: Interactive Donut chart for cost visualization.
- **Detailed Reports**: Monthly and annual cost projections.
- **Dark Mode**: Sleek, professional UI.

## Local Development

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Deployment

This project is configured for deployment on GitHub Pages using GitHub Actions.
